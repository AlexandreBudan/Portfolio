Article 4: Conducting a conference for LMDW

The production of a conference for Les Mystères du Web on today's Open Source tools.

Continue reading...

To beginning

Explanation of the event

What was the theme?

This event organized in partnership with the TUBA Lyon institute had as its themes the open source tools in the digital world today. The goal was to present to some actors of the world of work the possibility of changing their tools for free and sometimes more complete open source tools.

The presentation was therefore open to all and about ten people came to attend.

Preparation of the event


How i said in the last article, "Everything is a documentation

So the preparation has one too

We were thus responsible for following the preparation of the event based on documentation that had been produced during the last event.

Once all this was prepared in bags, we left on foot to join the event room (being less than 200 meters from the premises)

Once arrived on site I had to install the equipment such as the projectors, the different computers and the projectors with the constant pressure that if something did not work according to Hugues (the boss), we would be told off.

To conclude


This event taught me several things about open source tools, including several that are very interesting to implement.

However, the difficulty in using these tools is often such that software can be used by developers, so I think it would be complicated to include this within a company that has people who are not very comfortable with computers.

It was still very enriching and I have a positive memory of it.
