PrendsTaGo - Github

Description

PrendsTaGo is a 2nd-year graduate project at Ynov Campus Lyon. We were asked to create a functional car rental site where you could reserve a car, view its features, and many other constraints. The aim of this project was to learn php from scratch, but also for me to learn more about Docker. I really enjoyed doing this project, as I'm already very comfortable working with php frameworks. I was also able to deal with a lot of Docker scenarios, which meant that I gained a lot of skills on this project.

GAVE - Github

Description

GAVE is a 1st year graduate project at Ynov Campus Lyon. We were asked to create a music website where you could find information on artists, see their upcoming concerts, etc... This project enabled us to discover what an API was and how to use it, notably on a Golang server. As I was already familiar with the concept of APIs in php, this project was a good way of refining what I already knew and learning more about the Golang language.

Basket Joyride - Github

Description

Basket Joyride is a 2nd year graduate project at Ynov Campus Lyon. We were asked to create a 2D or 3D game, respecting the constraints that had to be present in the game. This project enabled me to discover what Unity is and to learn how to use it to make this 2D game. In particular, we had to add a life and death situation, a monster chasing us, a money system, etc... I particularly enjoyed doing this project because it was the first time I'd been involved in something I knew nothing about.

Description

TCP Innovation, founded in 2007, is an IT company that offers services for the creation of websites, from graphic charters to the detailed and unique realization of websites.

Les Mystères du Web, founded in 2019, is a for-profit platform managed by TCP Innovation, with the aim of making available a range of articles, conferences and more, so that everyone can find out more about today's IT and how it works.

It was in this context that I carried out my end-of-second-year internship as a web developer. My tasks included php from scratch, python scripting and the use of CMS such as Jumla. I was also able to carry out tasks for Mystères du Web, such as a conference on today's Open Source tools.
